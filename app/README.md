# Google 翻譯語音播放按鈕 (Chrome 外掛示範)

這個示範專案已經升級！目前在畫面上會出現包含**四個按鈕**的控制面板，可以分別針對「原文」與「翻譯結果」進行無限循環播放與停止。

## 功能介紹
- **🔁 循環播放(原文) / (翻譯)**：點擊後，外掛會自動尋求目標區塊的音檔。當音檔播放完畢時，會每秒自動偵測並重新點擊播放，達成語言學習或反覆聆聽的「循環播放」效果。
- **⏹️ 停止(原文) / (翻譯)**：隨時中斷循環播放。

## 檔案結構與說明
1. **`manifest.json`**: 外掛的核心設定檔。
2. **`content.js`**: 核心邏輯。其中設計了計時器（`setInterval`），每秒檢查 `aria-pressed` 或是按鈕上的狀態，一旦發現停止播放，就會立刻再次觸發原生的點擊事件。
3. **`content.css`**: 美化這四顆按鈕的控制面板。

## 如何更新或重新載入外掛？

因為我們剛剛修改了程式碼，如果你之前已經安裝過，你需要讓 Chrome 重新讀取最新的程式：
1. 回到 `chrome://extensions/` 頁面。
2. 找到這個外掛。
3. 點擊卡片右下角的 **「重新載入 (Reload)」** 按鈕（長得像一個重新整理的圓圈箭頭圖示）。
4. 然後回到 Google 翻譯網頁 **重新整理頁面 (F5)**，你就會看到新的包含四個按鈕的漂浮控制視窗了！
